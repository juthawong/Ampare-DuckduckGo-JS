# Ampare-DuckduckGo-JS
An Super Easy Javascript String Framework To Access DuckDuckGo Public API


## How To Use
  Just Chain DuckDuckGo Method After Any String. It will return an Object.

### Example
```javascript
   "Facebook".DuckDuckGo();
   "Github".DuckDuckGo();
````
